# L05
Python序列與迴圈
## 課程筆記
這一次就稍微學了一下迴圈的部分，例如for迴圈:
```
for i in range(a): #就是0-a
    #你想要他重複的事情
```
當然也可以range()那邊直接放一個array，他就會一個一個跑裡面的東西，另外還有while 迴圈:
```
n = 1
while n!=0 :
    #這樣就是n不等於0都會執行
```
如果你希望她無限執行，就寫while True:，另外迴圈裡面是可以繼續包迴圈的，例如:
```
for i in range():
    for j in range():
```
這樣就會重複地跑裡面的迴圈了。

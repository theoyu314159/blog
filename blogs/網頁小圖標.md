# 網頁小圖標
HTML

## 圖標
很多人寫網頁都可能只有學到
```
<title>標題</title>
```
然而並沒有學到圖標怎麼放，但這短短一行，卻能有效增加美感。
![example](https://i.meee.com.tw/NqPM8eV.png)

## 實作
首先準備好一張圖片，接著上傳到圖床(這樣比較不占空間)，如果大家沒有用圖床習慣，我蠻推薦這個的:
[meee免費圖床](https://meee.com.tw/)
接著開啟HTML檔案，在tittle下面加上一行:
```
<title>title</title>
<link rel="icon" href="url" type="image/png">
```
這樣就完成囉!是不是很簡單呢。

# 二進位
探討二進位
# 哪裡會用到
今天我們要來談論一個在電腦中非常重要的東西，它叫做二進位，他常用於電腦電路等方面上，簡單來說就是用0, 1來表示，舉例來說，例如電腦開關表示，一個是0, 一個是1。
# 二進位
通常我們學校教的數學都是10進位，所以很理所當然的11就是10, 1組合而成，而二進位的11則是1, 2組合而成，所以以10進位來說，二進位的11就是3，他的進位圖大概像這樣，會依著1, 2, 4, 8, 16, 32...下去:

# 計算
如果我們每一次都要畫出來計算也太麻煩了吧，所以我們選擇更好的方法計算，就是將數字不斷的用2除下去，並記下他的餘數，拿21來舉例好了:
21 / 2 = 10...1
10 / 2 = 5...0
5 / 2 = 2...1
2 / 2 = 1...0
1 / 2 = 0...1 (要記得算到0為止喔)
好的這樣我們就知道21的2進位會是10101

# 佇列

## 資料結構
資料結構是程式考試等等時常會遇到的東西，而佇列也是其中一個重要的東西，今天來稍微簡單的描述一下。

## 原則
我們來想像一下:
有一台火車要過山洞，
總共有4節車廂，
1是車頭4是尾，
1先進去，
接著2, 3, 4，
誰先出來呢
1, 2, 3, 4
![pic](https://i.meee.com.tw/zgPEy5R.png)
(圖很醜對不起)
這就是他保有的'first in, first out'的原則。

## 實作
我們來簡單用Python實作一下:
```
from collections import deque

queue=deque()

queue.append('a')
queue.append('b')
queue.append('c')
print(queue)

queue.append('d')
print(queue)

print(queue.popleft())
```
我們現import deque，接著將a, b, c,放入這一個queue，接著放入d，最後再將他取出，照著上面執行你會得到:
deque(['a', 'b', 'c'])
deque(['a', 'b', 'c', 'd'])
a

# L04
Python序列
## 課程內容
今天大概在講array這一部分，首先如果你要訂一個序列，就這樣:
```
list1 = [1, 2, 3, 4]
```
當然[，]後面不一定要空格啦，個人習慣看起來舒服，啊裡面可以同時存有字串和數字，例如:[1, 2, 3, "A", 5]，接下來學了一些區間:
```
print(list1[1:4])#1-3
print(list1[1:])#1-尾
print(list1[:3])開始-2
print(list1[:])全部
print(list1[::-1])#倒著算
print(list1[::2])#每隔2個取一個
```
接著如果要在array裡面新增東西有兩個方法:array.append(), array.extend()，要拿出東西就array.pop()或是del array[], array.remove()，最後還有一些實用技巧:
```
array.sort()#排序
array.reverse()#反過來
```

# 堆疊
stack

## 資料結構
相信各位如果有在準備一些程式相關考試一定對堆疊(stack)感到不陌生，那今天就稍微簡短的介紹一下他。

## 原則
基本上我們可以直接用字面上去理解，舉例來說:
如果你把一大堆書疊起來，你只能從最上面移動，不燃整疊可能會倒，所以你越晚疊上去的書如果要拿就越早拿到，而一開始就在的書，就要最後才能拿到。
![很醜的圖片](https://i.meee.com.tw/YeQ3r9G.png)
(畫很醜抱歉)
這就是堆疊'last in first out'的原則。

## 實作
接著我們用python實作一次:
```
stack = ['a', 'b', 'c']
print(stack)

stack.append('d')
print(stack)

stack.pop()
print(stack)

stack.pop()
print(stack)
```
我們簡的使用python做了讓他把'd'放到資料最後面，接著拿出後面兩個東西，執行後，你應該得到如下:
['a', 'b', 'c']
['a', 'b', 'c', 'd']
['a', 'b', 'c']
['a', 'b']

# 如何灌Raspberry Pi Zero
linux

## Raspberry Pi Imager
首先我們先到官網下載'Raspberry Pi Imager'。
[官網](https://www.raspberrypi.com/software/)

## 灌
接著進去後就可以開始灌了:
1 device : raspberry pi zero(依照自己的板子選)
2 os : zero建議選無桌面不然很慢...
![選](https://i.meee.com.tw/N1uZnz9.png)
3 空間 : 就選你要的sd卡吧
4 hostname : 主機名可以自己決定
5 地方 : 看你住哪裡囉
6 user : 自己取名子
7 wifi : 如果不是W可以不用輸入 是的話就輸入吧
8 ssh : 建議開一下
9 pi connect : 方便的話開一下比較保險

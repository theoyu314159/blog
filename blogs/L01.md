## 內容?
最基本的'Hello world'，接下來他告訴我可以加end=""來改結尾:
```
print(“Hello World!, end = "")
```
再來我學到了input()，一開始他會是字串，需要自己改成int(), float()，我知道這些聽起來很簡單我也都會，但基礎課程還是紀錄一下。
接下來我學到了大括號，我也不知道怎麼叫，反正就是可以在print()的時候直接插入?我示範給你們看好了:
```
#假設要印出a + b
a = 1
b = 2
print(f"答案是{a+b}") = print("答案是"+int(a+b))
```
再來就是一些基本運算，/是除法，*是乘法，**是次方，//取除法整數，%取餘數，喔還有，我示範一個東西:
```
print(8/3:.2f)
```
這樣他就會取道小數點後兩位，這是我這堂課新學到的東西。

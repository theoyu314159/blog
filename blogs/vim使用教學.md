# vim使用教學
linux

## nano
原本linux預設應該是使用nano來編輯檔案，然而其實還有另一款好用的工具叫做vim。
![nano](https://i.meee.com.tw/NfBkd9v.png)

## 下載
這一步很簡單，平常有用linux的人應該都會:
```
sudo apt install vim
```

## 開始使用
首先你會需要先'vim filename'來開啟一個新檔案，接著你會發現甚麼東西都無法輸入，這時按下i，就可以開始輸入了，接著如果要離開請先按esc跳出輸入模式，接著你有三個選擇:
':q' : 離開
':w' : 存檔
':wq' : 離開又存檔
如果你繼續待著，你想要控制游標:
'h' : 左
'j' : 下
'k' : 上
'l' : 右
'gg' : 最上面
'G' : 最下面
'0' : 最前面
'$' : 最後面
'x' : 刪字
'dd' : 刪行
'yy' : 複製一行
'p' : 貼上
'/ keyword' : 尋找
大概新手會用到就這樣囉!
